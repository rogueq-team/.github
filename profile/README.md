# BrandConnect

Платформа для прямого взаимодействия рекламодателей и владельцев площадок  (блогеров, медиа, сайтов) без посредников.  
README описывает назначение проекта, архитектуру, технологический стек и порядок запуска.

---

## Цель
Упростить и автоматизировать процесс поиска, согласования и заключения рекламных сделок между рекламодателями и площадками без участия агентств и высоких комиссий.

---

## Задачи
- Поиск рекламных площадок и рекламодателей в одном сервисе  
- Создание и отклик на рекламные заявки  
- Ведение сделок и общение сторон в реальном времени  
- Централизованное управление рекламными кампаниями  

---

## Технологический стек

### Backend — brandconnect-backend
- ASP.NET Core Web API (C#)
- Архитектура: REST API
- Взаимодействие: WebSocket (чат)
- Аутентификация и авторизация: JWT
- Безопасность паролей: Hash + Salt
- Документация API: Swagger

### Frontend — brandconnect-frontend
- React
- Взаимодействие с backend: REST API + WebSocket

### Data
- PostgreSQL — основная база данных
- Хранение пользователей, заявок, сделок, сообщений

### Инфраструктура
- Docker + Docker Compose (dev-окружение)
